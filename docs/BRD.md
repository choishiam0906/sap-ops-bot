# BRD (Business Requirements Document)
# SAP 운영 자동화 AI 봇

## 1. 비즈니스 목적

### 프로젝트 배경
GV_MM팀에서 축적한 SAP 운영 노하우("운영업무 시간절약 팁모음", 38슬라이드)가 PPT 형태로만 존재하여,
지식 공유와 활용에 한계가 있다. 이를 AI 봇으로 자동화하여 운영 효율성을 높인다.

### 비즈니스 가치
1. **운영 효율성 향상**: T-code 검색/절차 확인 시간 50% 절감
2. **지식 보존**: 개인 종속 지식을 조직 자산으로 전환
3. **온보딩 가속**: 신규 인력 학습 기간 단축 (3-6개월 → 1-2개월)
4. **오류 대응 속도**: 적절한 디버깅 도구 즉시 안내

## 3. 비즈니스 요구사항

### BR-1: 자연어 기반 SAP 운영 질의응답
- **설명**: 사용자가 한국어 자연어로 SAP 운영 관련 질문을 하면, AI가 적절한 답변을 제공
- **비즈니스 규칙**: 답변에는 항상 관련 T-code, 실행 절차, 주의사항이 포함되어야 함
- **수용 기준**: 기존 PPT에 있는 내용에 대해 80% 이상 정확한 답변

### BR-2: Microsoft Teams 통합
- **설명**: 별도 앱 설치 없이 Teams 내에서 바로 사용 가능
- **비즈니스 규칙**: Copilot Studio를 통한 기업 표준 배포
- **수용 기준**: Teams 채팅에서 '@SAP운영봇' 멘션으로 즉시 사용

### BR-3: 지식 베이스 관리
- **설명**: 관리자가 SAP 운영 지식을 추가/수정/삭제할 수 있는 관리 도구
- **비즈니스 규칙**: 승인된 관리자만 접근 가능
- **수용 기준**: 지식 추가 후 5분 이내 챗봇 응답에 반영

### BR-4: 사용 현황 모니터링
- **설명**: 봇 사용 현황, 자주 묻는 질문, 미답변 질의를 파악할 수 있는 대시보드
- **수용 기준**: 일별/주별 사용 통계, 인기 질문 Top 10 제공

## 5. ROI 분석 (추정)

### 비용 절감 효과
- SAP 운영자 1인 기준 T-code 검색/절차 확인: 일 30분 → 15분 (50% 절감)
- 운영자 10명 기준: 월 약 **50시간** 절감
- 신규 인력 온보딩: 건당 약 **80시간** 절감 (연 2-3명)

### 투자 비용
- 초기 개발: 4주 (개발 인력 1명)
- Azure OpenAI API: 월 약 $50-100 (사용량 기반)
- 인프라: Azure App Service 기본 플랜

### 예상 ROI
- 손익분기점: 런칭 후 약 2개월
